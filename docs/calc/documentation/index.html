language

functions

currently there are 2 types of function things in cjs, a host declaration and a regular function

host declarations

a host declaration lets you tell the compiler how to call a host function, which lets you do things like write to the output

they look like this 

host {{type}} {{name}}({{arguments}});

for example you could do

host int notarealhostfunction(int a, int b);

once youve declared them you can call them normally

list of host functions

_private_put

writes a number to stdout. will be replaced with something better.

int _private_put(int value);

regular functions

functions can be declared similarly to other languages, you put the type then name and arguments then a block

example:

int add(int a, int b) {
	return a + b;
}

then you can call them

variables

a variable is declared like this

{{type}} {{name}} [[= {{initializer}}]];

some examples:

int thing;
int hi = 1;
